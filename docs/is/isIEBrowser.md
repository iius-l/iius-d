# isIEBrowser

判断当前是否 IE 浏览器，不设计到对 Edge 浏览器的判断




### TypeScript 类型声明

```typescript
function isIEBrowser (): boolean
```



### 添加版本

`0.0.18`



### 参数

无需传入参数




### 返回

`boolean`



### 例子

```typescript
const isIE = isIEBrowser()
// true | false
```
